### 問 2

```
let moji = prompt('年齢は？');
let age = parseInt(moji);
if( age <= 5 && age => 65 ) {
  console.log('幼児と高齢者');
}
```

プロンプトで「年齢は？」と表示させ、入力した結果を変数 moji に代入
変数 moji を整数化して変数 age に代入
変数 age が 5 以下、65 以上の場合、「幼児と高齢者」をコンソールに表示
=> true になることがない。
